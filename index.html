Current Research

SimCLR Stability Research

Investigating stop-gradient asymmetry and temperature scheduling to reduce loss variance and stabilize gradients on CIFAR-10/ISL datasets. Working on reproducible ablations to improve self-supervised learning performance.

Indian Sign Language Recognition

Developed deep learning systems for Indian-Gujarati Sign Language recognition, focusing on dynamic gesture classification using enhanced LSTM architectures with custom attention mechanisms. Created datasets and evaluation frameworks for lower compute/storage requirements.

Student Knowledge-Base Chatbot

Building an AI-powered chatbot using retrieval-augmented generation (RAG) with FAISS for answering student queries about course syllabi and academic content. Prototype development using Gradio/FastAPI.
